# 智能体协作新方案（身外化身）

## 两种模式

- 唯一真身模式：只用唯一的智能体，进行对话、调用工具
- 身外化身模式：唯一真身进行任务规划，身外化身来进行具体的任务执行

## 唯一真身

没有特殊的，就是单agent模式

## 身外化身

两种角色的Agent：

- Main Agent（唯一真身），目的是做任务规划，和人类进行沟通，完善任务书
- Sub Agent（身外化身），由Main来根据任务规划书，安排Sub干活
- 身外化身在创建时，具有和Main一样的上下文，干完活后销毁，合并到Main
- 所有Agent共享记忆

协作文件：

- 任务书：Main生成和维护
- 任务单：Main交给Sub
- 回执单：Sub完成回复Main
- 任务日志：Main和Sub共同维护，记录任务经验，共同规避错误方法
